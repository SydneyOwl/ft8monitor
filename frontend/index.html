About:

This page shows some statistics on 6m band FT8 signal, and the antenna is a simple dipole on the balcony in Chengdu.

The upper histogram shows the number of decoded messages received in a time period, with the color shade representing the average SNR value. A number of -1 indicates that the monitor station is temporarily offline.

The world map below shows the distribution of signals in each country during the time period selected in the upper window (according to callsign).
